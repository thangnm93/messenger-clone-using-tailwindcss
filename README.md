## Demo

On Vercel [https://nextjs-messenger-clone.vercel.app/](https://nextjs-messenger-clone.vercel.app/)

## Install

```bash
npm install
# or
yarn install
```

## Run

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
